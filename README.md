# Hercules App - Codebase 🔱

Repository ini berisi codebase untuk aplikasi **Hercules**, yang digunakan untuk pengelolaan layanan dan treatment di **Deanna Day Spa**.

## 🛠️ Tech Stack
Aplikasi ini dibangun menggunakan teknologi modern:
* **Frontend:** React + Vite (TypeScript)
* **Backend:** Convex
* **Styling:** Tailwind CSS (index.css)

## 📁 Struktur Folder Utama
* `/src`: Berisi komponen utama dan halaman aplikasi.
* `/convex`: Pengaturan backend dan skema database.
* `/public`: Aset gambar dan file statis lainnya.

## 🚀 Cara Menjalankan (Local)
1. Clone repository ini.
2. Jalankan `pnpm install` untuk menginstal dependensi.
3. Jalankan `pnpm dev` untuk memulai server lokal.
4. Pastikan environment Convex sudah terhubung.
